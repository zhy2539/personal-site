## 为什么做这个站

这是一个轻量、可维护的中文个人站点，聚焦三件事：

1. **技术博客** — 用 MDX 写文章，记录实践
2. **MCP 目录** — 策展优质 Model Context Protocol 服务器
3. **Skill 目录** — 整理 Cursor / Agent Skill 资源

## 技术选型

- Next.js 15 App Router + TypeScript
- Tailwind CSS 响应式 UI
- JSON 文件存储 MCP/Skill 数据（零数据库运维）
- Vitest + Playwright 测试保障质量

## 持续更新

站点内置 **Loop Engineering** 机制：每周自动发现新的 MCP/Skill 候选，生成 PR 供人工审核，Admin 后台可手动增删改。

欢迎探索 [MCP 目录](/mcp) 与 [Skill 目录](/skills)！
